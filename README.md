# Web-Scraping-Mercado-livre

🛒 Web Scraping - Mercado Livre
Projeto desenvolvido em Python para realizar coleta automatizada de dados (web scraping) no site do Mercado Livre. A aplicação captura informações de produtos como título, preço e link, organizando os dados para análise posterior.

🎯 Objetivo
O principal objetivo deste projeto é aplicar conceitos de web scraping e manipulação de dados, além de desenvolver habilidades práticas em automação e coleta de informações na web.

🚀 Tecnologias utilizadas
Python
Requests
BeautifulSoup
Pandas

⚙️ Como executar o projeto
Clone o repositório:
git clone https://github.com/seu-usuario/seu-repositorio.git
Acesse a pasta do projeto:
cd seu-repositorio
Instale as dependências:
pip install -r requirements.txt
Execute o projeto:
python main.py

📊 Funcionalidades
🔍 Busca automática de produtos no Mercado Livre
💰 Coleta de preços dos produtos
📝 Extração de títulos e links
📁 Organização dos dados em tabela (Pandas / CSV)
📷 Exemplo de saída
Produto: Notebook Dell Inspiron
Preço: R$ 3.200
Link: https://www.mercadolivre.com.br/...

⚠️ Observações
Este projeto foi desenvolvido para fins educacionais
O site pode limitar requisições automáticas (anti-bot)
Recomenda-se utilizar intervalos entre requisições para evitar bloqueios
